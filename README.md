# Line-of-Text

People who enjoy taking notes in the most effective method would benefit from this program. It might be the ideal tool for making a secret diary that no one else would have considered.

# Programming Language Used
Python

# How to run this program
1. Use the command line or a Python IDE to run the application.
2. Enter your text
3. The user will be asked to repeat the procedure. if Yes, the procedure goes back to step 2. If not, the application terminates and the terminal will guide you on how to see the file path of your written message.
